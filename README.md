# 🛡️ Verity AI

An AI-powered Fake News Detection platform that analyzes news articles using multiple Machine Learning and Deep Learning models.

## 🚀 Live Features

- Detect Fake vs Real News
- Multi-model predictions
- Confidence scores
- Beautiful React dashboard
- Search history popup
- User authentication
- Modern UI/UX

---

## 🧠 Models Used

- Logistic Regression
- LSTM Neural Network
- DistilBERT Transformer
